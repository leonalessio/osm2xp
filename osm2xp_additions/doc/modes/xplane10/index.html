Xplane10 mode

Author : Benjamin Blanchet

Website : http://osm2xp.com

Email : osm2xp@gmail.com

Xplane10 mode generates building, landmarks and forests.

Buildings are generated using facades objects, which are simple textured polygons.

You can use osm2xp rules to place more complex 3D objects using osm data.

Remember that xplane is currently a 32 bit application, and can crash id there is too much objects on a tile.

Try to lower your settings if xplane crashs at startup.
