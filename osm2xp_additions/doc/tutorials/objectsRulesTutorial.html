Generation rules tutorial

This page is a work in progress and will be updated in next version.


Generation rules are an osm2xp feature that let you customize what is generated by the software.
It's a very powerfull feature  , yet very simple to use.

The concept is to link an openStreetMap tag with some xplane objects (forests, 3d objects, facades).

This way, you can tell osm2xp to generate for example a church 3d object each time the tag "amenit=place_of_worship" is encountered.

Generation rules panels let you export and import your settings to xml files, so you can share them with other users.
