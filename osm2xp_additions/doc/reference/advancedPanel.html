Advanced panel reference

The advanced panel lets you set a few misc options.
It's composed of 2 sections:

  1. Output format
  2. Stats options


1.Output format


You can choose in this section the output format osm2xp will use.
Right now the available options are:
-DSF (xplane format)
-OSM (openStreetMap format)

The OSM format is mainly for debug purpose, but can be usefull to extract a tile from a bigger file.

2.Stats options




-Generate XML stats : if checked , osm2xp will export stats for each tile in a xml file.
-Generate PDF stats : if checked , osm2xp will export stats for each tile in a pdf file, with charts.
